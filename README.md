# Game Package Management App

## 概要

ゲームパッケージを管理するWebアプリケーションです。

ゲームタイトル、プレイ状況、レビューなどを登録・管理できます。

## 使用技術

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express

### Database
- MySQL

## 機能

- ゲーム一覧表示
- ゲーム情報登録
- ゲーム情報編集
- ゲーム情報削除
- 画像アップロード
